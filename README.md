| Linux | x86_64 | ✔️ Primary For Testing | ✔️ | Update-Linux |

| MacOS | Apple Silicon (aarch64) | ✔️ *<sup>1</sup> | ❌ | Update-MacOS |
| Other | Any | ❌ | ❌ | N/A |
